# UI Structure

The frontend uses a Google Drive-like productivity interface without Google branding.

## Primary Layout

- Top bar: search, upload/create actions, account menu.
- Left sidebar: My Drive, Shared with Me, Recent, Starred, Trash, Admin.
- Main content: list/grid file browser.
- Breadcrumb: current folder path.
- Right panel: details, activity, permissions, AI summary.

## Core Interactions

- Drag-and-drop upload.
- Multi-select with bulk actions.
- Right-click context menu.
- Sharing dialog.
- File preview modal.
- Upload progress drawer.
- Sort/filter by name, size, type, modified date.

## Accessibility

- Keyboard navigable file grid/list.
- Visible focus states.
- ARIA labels for menus/dialogs.
- Responsive layout for tablet and phone.
